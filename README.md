# 🦕 Jurassic Meme Generator

**Substitua o rugido do dinossauro por qualquer áudio e baixe o meme pronto.**

👉 **[Abrir o site](https://jocowski.github.io/jurassic-meme-generator/)**

---

## Como funciona?

1. Acesse o site
2. Envie qualquer áudio: grito, música, voz, áudio de WhatsApp...
3. Clique em **Gerar Meme**
4. Assista o resultado e baixe o vídeo

Tudo roda direto no navegador — sem upload para servidor, sem cadastro.

**Formatos aceitos:** MP3, WAV, M4A, OGG

---

## Rodando localmente

```bash
npm install
npm run dev
```

Acesse `http://localhost:5173/jurassic-meme-generator/`.

---

## Stack

- [React](https://react.dev/) + [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [FFmpeg.wasm](https://github.com/ffmpegwasm/ffmpeg.wasm) — edição de vídeo no navegador
